# osa 4

## Blogilista

**installed:**

- express
- cors
- mongoose
- nodemon (--save-dev)
- dotenv
  <br />

**scripts:**

```js
"start": "node index.js"
```

can be run

```shell
npm start
```

<br />
```js
"dev": "nodemon index.js"
```
can be run 
```shell
npm run dev
```
(full path would be node.modules/.bin/nodemon)
